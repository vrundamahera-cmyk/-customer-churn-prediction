# Customer Churn Prediction 📉

End-to-end ML project predicting customer churn using Python, scikit-learn & Streamlit.

## Features
- Random Forest & Logistic Regression models (81% accuracy)
- Interactive Streamlit dashboard with 4 tabs
- Live single-customer churn prediction
- EDA, confusion matrix, ROC curve, feature importance

## Run Locally
```bash
pip3 install -r requirements.txt
python3 -m streamlit run app.py
```

## Tech Stack
Python · Pandas · scikit-learn · Streamlit · Matplotlib · Seaborn
